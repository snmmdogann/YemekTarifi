# YemekTarifi Projesi

Bu proje iki ana bölümden oluşmaktadır:

## 1. API Projesi
Kullanıcı, tarif, yorum, favori ve planlama işlemlerini yöneten backend katmanıdır.

## 2. Mobil Uygulama
.NET MAUI ile geliştirilmiş kullanıcı arayüzüdür. API ile haberleşerek tarif listeleme, favorilere ekleme, haftalık plan oluşturma gibi işlemleri gerçekleştirir.

## Kullanılan Teknolojiler
- C#
- .NET 8
- ASP.NET Core Web API
- .NET MAUI
- XAML
- Visual Studio

## Proje Yapısı
- `YemekTarifi.API` → backend
- `YemekTarifi.App` → mobil uygulama
